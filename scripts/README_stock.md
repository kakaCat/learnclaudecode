# 股票查询工具集

本目录包含多个用于查询A股股票价格的Python脚本。

## 可用脚本

### 1. 五粮液详细分析报告
```bash
python wuliangye_stock.py
```
- 生成五粮液（000858.SZ）的详细分析报告
- 包含价格、技术指标、资金流向、基本面数据
- 输出格式化的分析报告
- 保存数据到JSON文件

### 2. 快速股票查询
```bash
python quick_stock_query.py
```
- 快速查询五粮液当前价格
- 简洁的输出格式
- 带颜色的涨跌显示
- 自动保存最新价格

### 3. 股票命令行工具
```bash
# 简单查询
python stock_cli.py 000858

# 详细查询
python stock_cli.py 000858 -d

# JSON格式输出
python stock_cli.py 000858 -j

# 保存到文件
python stock_cli.py 000858 -s output.json
```

支持的其他股票代码：
- `000858` - 五粮液
- `600519` - 贵州茅台
- `000001` - 平安银行
- `000002` - 万科A

## 数据说明

⚠️ **重要提示**：
- 当前脚本使用**模拟数据**生成股票价格
- 实际应用中应替换为真实的股票API
- 模拟数据基于基础价格加上随机波动生成
- 价格每分钟变化一次（基于时间种子）

## 扩展为真实API

要使用真实股票数据，可以：

1. **新浪财经API**（免费）：
```python
url = f"http://hq.sinajs.cn/list=sz000858"
response = requests.get(url)
# 解析返回的数据格式
```

2. **腾讯财经API**（免费）：
```python
url = f"http://qt.gtimg.cn/q=sz000858"
response = requests.get(url)
# 解析返回的数据格式
```

3. **专业金融数据API**（付费）：
- 聚宽（JoinQuant）
- 米筐（RiceQuant）
- Tushare
- Baostock
