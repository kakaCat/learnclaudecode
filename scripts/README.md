# Scripts 目录

开发和运维脚本集合。

## 🚀 启动脚本

### start_with_daemon.py - 带守护线程的启动脚本

**用途**: 开发环境启动，自动启动守护线程（心跳、监控、健康检查）

**使用方法**:
```bash
python scripts/start_with_daemon.py
```

**功能**:
- ✅ 启动心跳系统（每5分钟检查会话状态）
- ✅ 启动守护系统（每60秒监控资源使用）
- ✅ 启动健康检查端点（http://localhost:8000/health）
- ✅ 全局异常捕获（主逻辑崩溃时守护线程继续运行）
- ✅ 优雅关闭（Ctrl+C）

**架构**:
```
主进程
├── 守护线程 1: 心跳系统（后台线程）
├── 守护线程 2: 守护系统（后台线程）
├── 守护线程 3: 健康检查（后台线程）
└── 主线程: Agent 业务逻辑
```

**注意事项**:
- ⚠️ 守护线程运行在主进程内，主进程崩溃时守护线程也会停止
- ⚠️ 适合开发环境，生产环境建议使用独立守护进程或 systemd
- ✅ 主逻辑异常不会影响守护线程继续运行

## 📊 监控端点

启动后可访问：
- **健康检查**: http://localhost:8000/health
- **生命周期状态**: 通过代码调用 `get_lifecycle_status()`
