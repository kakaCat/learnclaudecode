# 心跳配置

## 会话保活

- 会话超时: 30 分钟无活动
- 心跳间隔: 每 5 分钟检查一次
- 自动保存: 每轮对话后保存状态

## 状态检查

- 检查项:
  - Session 是否有效
  - Bootstrap 文件是否加载
  - 记忆存储是否可用
  - 工具是否正常

## 错误恢复

- 轻微错误: 自动重试（最多 3 次）
- 严重错误: 记录日志并通知用户
- 状态损坏: 提示用户重新创建 session

## 资源清理

- 临时文件: 会话结束后清理
- 后台任务: 超时后自动终止
- 子 Agent: 任务完成后自动销毁

## 监控指标

- 对话轮数
- 工具调用次数
- 记忆写入/搜索次数
- 子 Agent 派发次数
- 错误发生次数
